Alert Shortcode

2020-10-22 1 分鐘閱讀 Shortcode用戶手冊Alert

本文展示瞭如何使用 alert shortcode。

{{< alert "Message" [type] >}}

The parameter type is optional. Default to info.

Info

*[^_^]:# {{< alert "Info" >}}

{{< alert “Info” >}}

Success

{{< alert "Success" success >}}
Success

Warning

{{< alert "Warning" warning >}}
Warning

Danger

{{< alert "Danger" danger >}}
Danger
